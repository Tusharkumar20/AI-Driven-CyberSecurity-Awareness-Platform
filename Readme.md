# Instructions

Place the .env file(188 bytes) in Frontend  , .env file (262 bytes) and .json file in backend  that are shared with you in the respective folders.

Node js must be installed before thesde steps.

Open two terminals for frontened and backend then execute the following commands on first run.
After that only "node server.js" and "npm run dev"

## Backend
cd frontend
npm install
node server.js

## Frontend
cd frontend
npm install
npm run dev

# After this open the local host link generated in the frontend terminal.
